perjalanan akademis dan organisasi 2022-2026

klub generation girl

tahun 2022 sebelum saya masuk ke bangku perkuliahan organisasi ini membantu saya berkembang dan menumbuhkan minat tambahan selain di bidang sains saya berpartisipasi membuat mini project tentang chatbot web dan api end point walau minat saya dulu bidang matematika dan biologi kegiatan ini bisa mengisi waktu luang selama liburan

R n D

selain itu saya juga mengikuti kegiatan yang diadakan oleh klub ini seperti seminar dan workshop yang berkaitan dengan teknologi dan pengembangan diri, saya juga aktif dalam diskusi kelompok untuk berbagi pengetahuan dan biografi dengan anggota lainnya. melalui organisasi ini saya bisa belajar banyak hal baru dan mengembangkan keterampilan yang berguna untuk masa depan saya.

gradio-chatbot "skincare"

untuk memenuhi tugas praktikum kecerdasan buatan dasar saya menggunakan gradio untuk integrasi pada web saya melalui API key, membuat pola percakapan dengan topik skincare dan web berjalan pada jupyter notebook

UI UX

sebelum membuat web kita perlu merancang erd, UML(seperti use case dan class diagram), user flow dan desain ui ux salah satu alatnya yaitu figma, dengan menggunakan figma saya bisa membuat desain web yang menarik dan mudah digunakan, saya juga bisa berkolaborasi dengan anggota kelompok untuk membuat desain yang sesuai dengan kebutuhan pengguna. melalui kegiatan ini saya bisa belajar banyak tentang desain web dan meningkatkan keterampilan saya dalam merancang antarmuka pengguna yang efektif.

petshop(praktikum web app)

pada praktikum web app saya dan kelompok membuat sebuah web dengan tema petshop menggunakan framework laravel, pada web ini terdapat fitur untuk menambahkan produk, menghapus produk, dan melihat daftar produk yang tersedia. saya menggunakan database mysql untuk menyimpan data produk dan menggunakan HTML, CSS, dan JavaScript untuk membuat tampilan web yang menarik dan PHP sebagai bahasa yang digunakan untuk proses backend.
